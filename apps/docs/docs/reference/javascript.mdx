本参考文献记录了Supabase的同构JavaScript库`supabase-js`中的每个对象和方法。

你可以使用 `supabase-js` 库来。

- 与你的Postgres数据库进行交互
- 监听数据库变化
- 构建登录和用户管理功能
- 管理大型文件
