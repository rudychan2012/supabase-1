本参考文献记录了Supabase的同构JavaScript库`supase-js`中的每个对象和方法。

你可以使用 `supase-js` 库来。

- 与你的Postgres数据库交互
- 听取数据库变化
- 调用Deno边缘函数
- 建立登录和用户管理功能
- 管理大文件
