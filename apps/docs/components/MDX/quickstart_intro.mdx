## 介绍

这个例子提供了建立一个基本用户管理应用程序的步骤。它包括。

- MemFire Cloud [Database](/docs/guides/database)：一个用于存储用户数据的Postgres数据库。
- MemFire Cloud [Auth](/docs/guides/auth)：用户可以用魔法链接登录（没有密码，只有电子邮件）。
- MemFire Cloud [存储](/docs/guides/storage)：用户可以上传照片。
- [行级安全](/docs/guides/auth#row-level-security)：数据受到保护，个人只能访问自己的数据。
- 即时[APIs](/docs/guides/api)。当你创建你的数据库表时，API将自动生成。

在本指南结束时，你将拥有一个允许用户登录和更新一些基本档案细节的应用程序。
